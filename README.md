# [Brunson](https://brunsonliu.github.io/)

武汉大学管理科学本科 → 中国科学技术大学管理科学与工程硕士（2026 入学）。专注组合优化、车辆路径问题与 Learning to Optimize。

---

## 我是做什么的

运筹学方向，主要做三件事：**建模**（把实际问题抽象成数学问题）、**求解**（精确算法 + 启发式 + 学习驱动）、**落地**（把算法用到真实的供应链和工业场景）。

本科毕设是分布式光伏电站的多无人机协同巡检——选址、分配、路径联合优化，MILP + MO-ACO-ALNS 混合算法，拿了校级优秀。参加过美赛（Finalist）、数维杯（Finalist）、运筹学会竞赛（队长，冷链仓网优化）。在长鑫存储做过运筹优化实习生（电路参数优化 + 产能规划），在丝宝集团做过数据分析实习生（投流优化 + 用户画像）。

## 这个网站是什么

我的个人主页。不是传统 portfolio，就是把我做过的事、学过的东西、想过的乱七八糟记录在这里。

**学术首页**：教育、研究方向、项目、竞赛、实习，点卡片能看详情。  
**学习笔记**：优化理论、机器学习、Linux、第一性原理……学过的随手记。  
**文学空间**：诗词、书、电影——好的句子舍不得忘掉。  
**运动**：跑步配速、篮球记录。  
**旅行**：去过的好地方。  
**思考**：随便写写的。

## 怎么加内容

都在 `app/data/` 里改 JS 文件。每个文件都有清晰的数组结构，复制一条改内容就行。

MDX 子页面（`app/study/`、`app/internship/`）直接写 Markdown。
